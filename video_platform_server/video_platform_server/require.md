# 1. 游客端

- 首页浏览视频
- 查看视频详情
- 点击其他功能提示登录

# 2. 客户端

## 个人页：

- 登录
- 注册
- 查看个人基本信息
- 修改个人基本信息
- 修改密码
退出登录
- 注销账号
- 发布视频
- 查看自己发布的视频
- 查看历史浏览
- 查看收藏的视频
- 签到获取硬币（必须有地方能够查看硬币数）
- 统计所有的发布视频数据，如点赞收藏投币等等

## 首页：

- 首页查看推荐视频，刷新
- 根据关键字搜索视频
- 查看视频详情，加入历史浏览
- 下载视频 
- 搜索视频使用分页搜索
- 点赞视频，取消点赞            
- 收藏视频
- 投币
- 发表评论（包括多级评论）
- 删除自己发布的视频下的评论
- 删除自己发的评论
- 点赞评论，取消点赞
- 举报视频

# 3. 管理端

- 登录
- 审核视频
- 封禁账号（被封禁的账号不能使用）
- 处理举报
- 导出所有用户信息表格
